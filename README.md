# POI Narration (MAUI)

## Mục tiêu
Ứng dụng thuyết minh tự động theo vị trí (GPS/Geofence) và kích hoạt bằng QR.

## Tiến độ 4 tuần
- Week 01: Setup + UI skeleton + chạy Android/iOS
- Week 02: Map + GPS foreground + highlight POI
- Week 03: Geofence + TTS/Audio + cooldown + log
- Week 04: QR + SQLite + polish + demo

## Chạy dự án
- Visual Studio 2022 + .NET MAUI workload
- Chọn Android Emulator / iOS Simulator
- Run (F5)
